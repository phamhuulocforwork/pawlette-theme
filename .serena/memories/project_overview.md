# Project Overview

## Purpose
This is a Pawlette theme package based on Catppuccin Mocha color scheme. It includes:
- GTK themes (GTK 3.0, GTK 4.0, Cinnamon, GNOME Shell, etc.)
- Icon theme (based on Tela-circle-dracula)
- Cursor theme (Bibata-Modern-Classic with hyprcursor support)
- Configuration files for various applications (bspwm, hypr, waybar, kitty, rofi, etc.)
- Wallpapers

## Tech Stack
- Shell scripts (bash)
- CSS for GTK themes
- RASI for rofi
- Various config file formats (INI, TOML, JSON, JSONC, CONF)
- Theme definition files

## Structure
- `configs/`: Application-specific config files for various WMs and apps
- `gtk-theme/`: GTK theme files (GTK 3.0, 4.0, Cinnamon, GNOME Shell, etc.)
- `icons/`: Icon theme with multiple sizes and cursor themes
- `wallpapers/`: Theme wallpapers
- `README.md`: Build instructions and documentation
- `LICENSE`: License file
