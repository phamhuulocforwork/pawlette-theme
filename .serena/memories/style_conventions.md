# Style and Conventions

## File Naming
- Use lowercase with hyphens for directories
- Config files follow application conventions
- Theme files use standard naming (index.theme, gtk.css, etc.)

## Code Style
- Shell scripts: Standard bash syntax
- CSS: Standard CSS formatting
- Config files: Follow respective application's format conventions

## Color Scheme
- Based on Catppuccin Mocha palette
- Primary accent: lavender
- Theme follows consistent color application across all components
